# Web Blog

Простой личный блог, реализованный на **PHP** с использованием микрофреймворка **Slim** и шаблонизатора **Twig**.

## Описание

Проект представляет собой минималистичный личный блог с упором на простоту архитектуры и читаемость кода.  
Используется микрофреймворк Slim без тяжёлых full-stack решений.

## Возможности

- Роутинг и HTTP-логика на базе Slim
- Шаблонизация представлений через Twig
- MVC-подобная структура (роуты → контроллеры → шаблоны)
- Чистый и понятный PHP-код
- Простое расширение функциональности
- Подходит как учебный или стартовый проект

## Технические особенности

- Slim используется как микрофреймворк, без избыточной абстракции
- Бизнес-логика отделена от представлений
- Централизованное описание маршрутов
- Соблюдение принципов KISS и DRY
- Проект легко расширяется:
  - подключение БД
  - добавление админ-панели
  - авторизация пользователей

## Назначение проекта

Проект может использоваться как:
- личный блог
- пример архитектуры Slim + Twig
- демонстрационный проект аккуратной PHP-архитектуры

## Стек

- PHP  
- Slim Framework  
- Twig
